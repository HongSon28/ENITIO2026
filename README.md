# Wanderer 2026 Code

## Features
- The code is updated from the 2022 code
- All of the outdated functions are updated to the newest version to ensure compatability with esp board v3

## Warnings
- The code is quite heavy, take **almost all** of the static memory of the board (~90%). This may affect future updates and improvements
- This code is **untested** on a real device. Further testing will be needed to ensure stability
- The features of the game remains at 2022 version, new updates are needed
- The virus infestation game features from 2022 code is currently disabled to save memory. To enable them, find the piece of code related to it which was commented out and include the bluetooth file.
